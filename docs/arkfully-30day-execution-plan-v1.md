# 아크풀리 30일 실행 플랜 v1

작성일: 2026-03-22
목표: 브랜드 메시지 정리, 홈페이지 초안 정리, 평일 매출 실험 시작

## 1. 이번 30일의 핵심 목표

- 아크풀리를 한 문장으로 설명할 수 있게 만들기
- 온라인 채널의 소개 흐름을 정리하기
- 화수목 평일 상품을 실제로 테스트하기
- 문의가 들어왔을 때 바로 응대 가능한 상태 만들기

## 2. 가장 먼저 집중할 것

이번 달에는 아래 3가지만 잡으면 충분하다.

1. 홈페이지 초안 마무리
2. 평일 라운지/스테이 상품 문구 확정
3. 인스타그램과 예약 링크 정리

## 3. 주차별 실행

### 1주차

- 홈페이지 문구 최종 확정
- 메인 대표 사진 최종 확정
- 조감도 문구 정리
- 스테이 / 라운지 / 대관 소개 문구 통일
- 예약 링크 구조 점검

완료 기준:
- 홈페이지에서 처음 보는 사람도
  - 아크풀리가 무엇인지
  - 스테이와 라운지가 어떻게 다른지
  - 어디로 예약해야 하는지
  바로 이해할 수 있어야 한다

### 2주차

- 평일 라운지 이용 공지 문구 확정
- 리셋 스테이 소개 문구 확정
- DM/문의 응대 템플릿 작성
- 자주 묻는 질문 10개 정리

완료 기준:
- 문의가 들어왔을 때 복붙해서 보낼 수 있는 답변 템플릿이 있어야 한다

### 3주차

- 인스타그램 프로필 문구 정리
- 하이라이트 구조 정리
- 피드용 게시물 6개 주제 기획
- 라운지 이용 가능일 공지 방식 결정

완료 기준:
- 인스타그램만 봐도
  - 어떤 공간인지
  - 어떻게 이용하는지
  - 예약은 어디서 하는지
  알 수 있어야 한다

### 4주차

- 평일 라운지 또는 리셋 스테이 테스트 오픈
- 문의 수, 저장 수, 클릭 수 기록
- 반응 좋은 문구/사진 확인
- 다음 달 보정안 정리

완료 기준:
- 실제 반응 데이터를 최소 1번은 받아봐야 한다

## 4. 상품 우선순위

### 1순위
평일 라운지 이용

이유:
- 진입장벽이 가장 낮다
- 평일 공실 대응에 직접적이다
- 스테이와 대관으로 이어지는 첫 경험이 된다

### 2순위
리셋 스테이

이유:
- 아크풀리의 가장 강한 매력을 잘 담는다
- 객단가가 높다
- 후기 자산과 연결하기 좋다

### 3순위
프라이빗 이벤트 / 소규모 대관

이유:
- 문의가 생기면 응대 가능하도록 준비하되
- 이번 달 핵심 실험은 평일 상품에 둔다

## 5. 운영 문구 예시

### 평일 라운지 공지
대관이 없는 평일에는 아크풀리 라운지를 1인 1만원으로 이용하실 수 있습니다.  
조용히 머물며 쉬어가고 싶은 분들을 위해 운영합니다.

### 리셋 스테이 소개
자연을 바라보며 천천히 쉬고, 읽고, 정리하는 시간을 위한 아크풀리만의 스테이입니다.

## 6. 채널별 역할

### 홈페이지
- 브랜드 소개
- 공간 구조 설명
- 예약 분기

### 인스타그램
- 무드 전달
- 운영 공지
- 첫 관심 유입

### 네이버 플레이스
- 신뢰 확보
- 기본 정보 확인
- 검색 대응

### 예약 링크
- 실제 행동 전환

## 7. 지금 필요한 최소 산출물

- 홈페이지 초안 1개
- 인스타 프로필 문구 1개
- 평일 라운지 공지문 1개
- 리셋 스테이 소개문 1개
- 문의 응답 템플릿 5개

## 8. 가장 중요한 체크포인트

- 소개 문구가 너무 길지 않은가
- 예약 경로가 헷갈리지 않는가
- 평일 고객이 이용 이유를 바로 이해하는가
- 사진이 공간의 결을 제대로 보여주는가

## 9. 다음 30일 이후 확장

이 30일이 끝난 뒤 반응이 좋으면 아래 순서로 확장한다.

1. 팀 오프사이트 안내 페이지 추가
2. 스몰웨딩/기념일 전용 소개 페이지 추가
3. 후기 기반 사회적 증거 강화
4. 유튜브 또는 짧은 무드 영상 활용
