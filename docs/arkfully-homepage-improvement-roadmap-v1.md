# 아크풀리 홈페이지 개선 로드맵 v1

작성일: 2026-03-22
목표: 아크풀리의 첫인상을 더 감각적으로 만들고, 온라인 노출과 예약 전환에 기여하는 홈페이지로 단계적으로 고도화한다.

## 1. 이번 1차 반영의 방향

- 메인 페이지를 정보 나열형에서 감도 중심 랜딩 구조로 재구성
- 자연, 조용함, 프라이빗함이 먼저 느껴지도록 대표 이미지와 카피 재배치
- 방문자 관점에서 `Stay / Lounge / Private Events` 선택 흐름을 더 빠르게 이해하도록 정리
- 후기 키워드와 공간 조감도를 활용해 신뢰 요소 보강

## 2. Step by Step 개선 순서

### Step 1. 메인 비주얼 고도화

목표:
- 첫 화면에서 "아크풀리답다"는 인상을 명확히 만든다.

할 일:
- 계절과 시간대 기준으로 대표 사진 3세트 선정
- 현재 hero 이미지 외에 봄/여름, 가을/겨울 대체 컷도 마련
- 사진마다 역할을 정한다
  - 첫인상 컷: 숲, 노을, 전경, 큰 창
  - 신뢰 컷: 객실 청결, 가구, 테이블, 디테일
  - 전환 컷: 실제 이용 장면, 세팅된 이벤트, 머무름의 분위기

선정 기준:
- 첫 1초에 풍경과 공간 성격이 읽히는가
- 수평/수직 구도가 웹 배치에 유리한가
- 밝기와 색온도가 다른 사진과도 조화를 이루는가

### Step 2. 카피 톤 정제

목표:
- 소개 문구를 더 짧고 강하게 만들어 브랜드 인지와 예약 전환을 동시에 잡는다.

할 일:
- 메인 헤드라인 후보를 3개 더 테스트
- 서브카피는 2문장 이내로 압축
- CTA 문구를 목적별로 분리
  - 스테이: 예약/숙박 중심
  - 라운지: 운영일 확인/가볍게 방문
  - 이벤트: 문의/상담 중심

추천 원칙:
- "좋은 공간" 같은 추상어보다 장면이 떠오르는 표현 사용
- 길게 설명하기보다 핵심 가치 2개만 또렷하게 전달
- 기독교 세계관은 배타적 문장보다 운영 태도와 분위기로 번역

### Step 3. 후기와 신뢰 요소 보강

목표:
- 감성만 있는 사이트가 아니라 실제로 믿고 예약할 수 있는 사이트로 만든다.

할 일:
- 실제 후기에서 반복되는 표현 10~20개 수집
- 후기 원문을 그대로 길게 붙이기보다 키워드형으로 먼저 구조화
- 가능하면 아래 항목을 별도 블록으로 추가
  - 평점 또는 후기 수
  - 재방문 이유
  - 청결/전망/친절 등 반복 키워드

데이터 소스:
- 에어비앤비 후기
- 네이버 플레이스 리뷰
- 인스타그램 DM/스토리 언급
- 운영자가 직접 받은 메시지

### Step 4. 페이지별 메시지 분화

목표:
- 메인에서 관심을 만든 뒤, 하위 페이지에서 구체적인 예약 판단을 돕는다.

우선순위:
- `stay.html`: 객실 구성, 인원, 가격대, 추천 이용 장면 보강
- `lounge.html`: 오픈일 안내 방식, 데이패스 성격, 개인 방문 장점 명확화
- `events.html`: 적합한 행사 유형, 수용 인원, 사진 사례, 문의 방식 정리
- `reservation.html`: 링크 모음 페이지를 넘어 "어디로 가야 하는지" 더 직관적으로 안내

### Step 5. SEO와 전환 요소 보강

목표:
- 검색 노출과 문의 전환을 함께 개선한다.

할 일:
- 페이지별 `title`과 `meta description`을 검색 의도에 맞게 세분화
- 이미지 파일명과 `alt` 텍스트를 검색 친화적으로 정리
- 본문에 지역명, 이용 목적, 공간 성격 키워드 자연스럽게 포함
- 예약 버튼 위치를 페이지 중간과 하단에 반복 배치

추천 키워드 축:
- 곤지암 스테이
- 광주 감성 숙소
- 프라이빗 대관 공간
- 자연 뷰 라운지
- 조용한 워케이션 공간

## 3. 사진 운영 체크리스트

- 너무 어둡거나 노출이 무너진 사진은 메인에서 제외
- 같은 장면의 유사 컷은 1장만 남기고 정리
- 사람 없는 컷과 머무름이 느껴지는 컷을 균형 있게 섞기
- 세로 사진은 모바일 우선 섹션에, 가로 사진은 히어로/갤러리에 우선 배치
- 계절감이 충돌하는 사진은 한 화면에 과하게 섞지 않기

## 4. 카피 운영 체크리스트

- 첫 화면: 1개의 강한 문장 + 1개의 설명 문장
- 본문: 브랜드 철학보다 이용 장면이 먼저 떠오르도록 작성
- CTA: "무엇을 누르면 어떤 행동으로 이어지는지" 분명하게 표현
- 후기 섹션: 추상 칭찬보다 실제 선택 이유를 중심으로 정리

## 5. 다음 추천 작업

가장 효율이 좋은 다음 순서는 아래와 같다.

1. `stay.html`, `lounge.html`, `events.html`를 메인과 같은 톤으로 확장
2. 실제 후기 기반 카피로 `Proof/Reviews` 섹션 교체
3. 사진 원본 중 대표컷을 다시 선별해 갤러리 재편성
4. 예약/문의 전환 경로를 더 짧게 줄이는 CTA 개편
