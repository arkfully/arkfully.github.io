# 아크풀리 홈페이지 와이어프레임 v1

작성일: 2026-03-20
목표: 브랜드 이해, 공간 구조 이해, 예약 전환

## 1. 홈페이지의 역할

아크풀리 홈페이지는 단순 정보 페이지가 아니라 아래 3가지를 동시에 해야 한다.

- 아크풀리가 어떤 브랜드인지 한 번에 이해시킨다
- 스테이와 라운지의 관계를 쉽게 설명한다
- 예약과 문의로 자연스럽게 연결한다

즉, 이 홈페이지는 "예쁜 공간 소개"보다 "왜 이 공간이 특별한지"를 먼저 전달해야 한다.

## 3. 메인 페이지 와이어프레임

### Section 1. Hero

#### 목적
첫 화면에서 브랜드의 결을 전달하고, 방문자가 "여기가 어떤 곳인지" 바로 감 잡게 한다.

#### 구성 요소
- 풀스크린 대표 이미지 또는 짧은 슬라이드
- 메인 헤드라인
- 서브카피
- CTA 버튼 3개

#### 추천 문구
- 헤드라인: 자연 안에서 조용히 쉬어갈 수 있는 당신의 아지트, 아크풀리
- 서브카피: 독립된 스테이와 라운지 공간을 통해 조용한 회복과 프라이빗한 머무름의 시간을 제공합니다.
- 버튼 1: 스테이 예약하기
- 버튼 2: 라운지 이용 안내
- 버튼 3: 대관 문의하기

#### 비주얼 방향
- 가장 넓은 자연 풍경 사진 1장
- 또는 큰 창 너머 숲이 보이는 실내 사진
- 첫 화면에는 정보보다 분위기가 먼저 보여야 한다

### Section 2. Brand Story

#### 목적
아크풀리의 가치와 철학을 짧고 명확하게 소개한다.

#### 레이아웃
- 좌측: 짧은 철학 문장
- 우측: 본문 2~3문단

#### 예시 구조
- 타이틀: 조용한 회복을 위한 공간
- 키워드 3개: 조용한 회복 / 자연과의 연결 / 프라이빗한 머무름
- 본문: 왜 아크풀리가 만들어졌는지, 어떤 사람에게 맞는지 설명

#### 메모
- 이 섹션에서 기독교 세계관은 직접적 종교 홍보보다 브랜드의 태도와 가치로 은은하게 드러내는 편이 좋다

### Section 3. Space Structure

#### 목적
스테이와 라운지의 관계를 한 번에 이해시킨다.

#### 레이아웃
- 상단: 공간 전체 조감도 일러스트
- 하단: 3개의 설명 카드

#### 카드 구성
- 카드 1: Stay
  - 상단의 독립된 공간
  - 최대 6인
  - 라운지 무료 이용 포함
- 카드 2: Lounge
  - 하단의 열린 공간
  - 대관 없는 날 개인 이용 가능
  - 1인 1만원
- 카드 3: Outdoor / Private Use
  - 야외 공간과 바베큐장
  - 촬영, 모임, 프라이빗 이벤트 가능

#### 중요 문장
아크풀리는 머무는 경험과 방문하는 경험이 함께 설계된 공간입니다.

### Section 4. Stay

#### 목적
숙박 공간의 감도와 차별점을 설명하고 예약으로 연결한다.

#### 레이아웃
- 좌측: 대표 사진 2~3장
- 우측: 소개문 + 정보 요약

#### 정보 요약 박스
- 최대 6인
- 침실 2
- 라운지 무료 이용
- 자연 속 프라이빗 스테이

#### 하단 CTA
- 스테이 상세 보기
- 예약 바로가기

#### 강조 포인트
- 전망
- 조용함
- 청결함
- 독서/사색/워케이션

### Section 5. Lounge

#### 목적
라운지가 "대관만 가능한 공간"이 아니라, 아크풀리 입문 경험이 가능한 곳임을 알려준다.

#### 레이아웃
- 상단: 넓은 창이 보이는 라운지 대표 이미지
- 하단: 2단 카드형 안내

#### 카드 A. 개인 방문
- 대관이 없는 날 개인 이용 가능
- 평일 중심 운영
- 1인 1만원
- 조용히 쉬거나 책을 읽고 대화하기 좋은 공간

#### 카드 B. 단체/대관
- 기업 행사
- 촬영 대관
- 소규모 워크숍
- 가족 및 기념 모임

#### CTA
- 이번 주 라운지 오픈일 보기
- 라운지 이용 문의

### Section 6. Private Events

#### 목적
대관과 스몰웨딩, 촬영, 기업 행사 수요를 따로 받아낼 수 있게 한다.

#### 레이아웃
- 타이틀 + 짧은 소개
- 4개 이벤트 유형 카드

#### 카드 예시
- Corporate Gathering
- Photo Shoot
- Small Wedding
- Private Celebration

#### 안내 포인트
- 너무 상업적이거나 소란스러운 행사보다 공간의 결에 맞는 프라이빗 이용 지향
- 시간당 대관 또는 맞춤형 문의 가능

### Section 7. Reviews

#### 목적
실제 고객 반응을 통해 신뢰를 만든다.

#### 레이아웃
- 짧은 후기 카드 3~6개
- 별점/키워드 요약

#### 추천 구성
- 전망이 좋다
- 청결하다
- 조용해서 쉬기 좋다
- 호스트가 친절하다

#### 메모
- 에어비앤비 후기, 네이버 후기, DM 후기 등을 짧게 큐레이션하면 좋다

### Section 8. Reservation

#### 목적
사용자가 헷갈리지 않고 예약 경로를 고를 수 있게 한다.

#### 레이아웃
- 3분할 카드 구조

#### 카드 1. Stay Reservation
- 숙박 예약
- 예약 버튼

#### 카드 2. Lounge Visit
- 라운지 개인 방문 문의
- 운영 가능일 확인

#### 카드 3. Private Events
- 대관/촬영/행사 문의
- 문의 버튼

#### 보조 정보
- 운영 시간
- 위치
- 주차 정보
- 문의 채널

### Section 9. Footer

#### 포함 정보
- 아크풀리 한 줄 소개
- 인스타그램
- 네이버 플레이스
- 유튜브
- 예약 링크
- 주소 또는 지역 정보
- 운영 시간

## 5. 모바일 기준 우선순위

모바일에서는 아래 정보가 가장 빨리 보여야 한다.

1. 메인 사진
2. 한 줄 소개
3. 스테이 / 라운지 / 대관 선택 버튼
4. 공간 구조 설명
5. 예약 버튼

모바일에서는 긴 설명보다 선택지가 먼저 보여야 이탈이 줄어든다.

## 6. 디자인 메모

- 색감은 숲, 흙, 나무, 햇빛을 떠올리게 하는 톤이 좋다
- 지나치게 호텔스럽거나 웨딩홀처럼 보이면 아크풀리다움이 약해진다
- 사진은 과한 보정보다 자연광과 실제 풍경의 설득력이 중요하다
- 조감도 이미지는 중간 섹션에서 매우 유용하다
- 스테이와 라운지는 반드시 시각적으로 분리해 보여줘야 한다

## 7. 첫 제작 버전에서 꼭 필요한 콘텐츠

- 대표 메인 사진 1장
- 스테이 사진 8~12장
- 라운지 사진 8~12장
- 야외 공간 사진 4~6장
- 후기 문구 5개 이상
- 공간 구조 조감도 1개
- 예약 링크 정리본

## 8. 실제 제작 순서 제안

1. 메인 페이지 단일 랜딩페이지부터 만든다
2. Stay, Lounge, Private Events는 앵커 섹션으로 먼저 운영한다
3. 이후 문의가 늘어나면 하위 상세 페이지를 추가한다

즉, 첫 버전은 작게 시작하고, 반응이 오는 섹션부터 깊게 확장하는 방식이 가장 효율적이다.
